=================
Project Extension
=================

This module was written to extend the functionality of projects. This module  creates new fields real_total, real_planned and real_effective that count total, planned and effective hours despite the fact if the task is in folded state in kanban. This module aswell adds priority-field to projects. In addition to that this module changes float-widgets to time-widgets in projects' and tasks' kanban views. 

Installation
============

Install the module form Settings->Local Modules
